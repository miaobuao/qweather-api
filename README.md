# QWeather API

A Python client for the QWeather API with location lookup capabilities and support for both synchronous and asynchronous operations.

## Features

- **Weather API Client**: Fetch current, daily, and hourly weather data
- **Location Lookup**: Find cities by English or Chinese names with fuzzy matching
- **Async Support**: All API calls available in both sync and async versions
- **Type Safety**: Full type hints and Pydantic models for all data structures
